# Python Section

This folder contains Python learning projects demonstrating fundamentals and advanced concepts.

## Projects

### 1. Chatbot (chatbot.py)
An interactive CLI chatbot demonstrating:
- Variables and operators
- Collections (list, dict)
- Loops (while, for)
- Functions
- Conditional statements (if/elif/else)

**Run:** `python chatbot.py`

### 2. Book Manager (bookmanager.py)
A book management application demonstrating:
- File I/O operations
- JSON serialization
- datetime handling
- Exception handling
- Type hints

**Run:** `python bookmanager.py`
